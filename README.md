In this project I designed an original programming language using YACC/BISON. The code is written in C++ and has the following parts:
- Syntax: Data types, Classes, Arithmetic and boolean expressions and Statements
- Symbol tables
- Semantic analysis
- Evaluation of arithmetic expressions and boolean expressions

Remarks:
- I do not use global variables of type AST   
- I use as few global variables as possible and avoid defining functions in the .y file
